About This Project

This single-page application was built for COMP 3512 Assignment 2. It uses only core JavaScript topics covered in lecture, including variables, conditionals, loops, arrays, objects, functions, DOM selection/manipulation, event handling, and JSON parsing.

  • Views: Home, Browse, Single Product, and Shopping Cart
  • Popup: About Us dialog
  • Technologies: HTML, CSS, JavaScript
  • Student: Tony Nguyen
  • Repository link: GitHub Repo

The design uses the provided product data and keeps the required functionality inside one HTML page. Header and footer stay consistent while the content in the middle changes between views.
